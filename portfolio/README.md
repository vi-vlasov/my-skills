# Portfolio — Detroit: Become Human style

Одностраничное портфолио, воспроизводящее главное меню Detroit: Become Human:
крупный портрет андроида с живой лицевой анимацией, LED-индикатор на виске,
горизонтальное меню внизу с синей плашкой выбранного пункта и уголками-скобками,
рассыпанная HUD-телеметрия, засвеченная белая студия с вертикальными
световыми панелями.

## Технологии рендеринга

- **three.js r170** (лежит локально в `vendor/`, страница полностью автономна)
- Физически корректные материалы (PBR), KTX2/Basis-текстуры, meshopt-компрессия
- Лицевая анимация на morph targets (52 ARKit-блендшейпа, живая запись
  моргания, бровей и речи) + взгляд, следящий за курсором
- HDR image-based lighting (PMREM + RoomEnvironment)
- ACES filmic tone mapping, sRGB output
- Постобработка: UnrealBloomPass (ореол света вокруг силуэта) + OutputPass
- LED-индикатор — emissive-геометрия с HDR-цветом, посаженная на кожу
  виска raycast'ом и пульсирующая в bloom
- Схема света: тёплый key + холодный fill + два rim сзади, «плавящие»
  контур в засвеченный фон

## Управление

- Наведение мыши / ←→ — выбор пункта меню, клик / Enter — открыть, Esc — назад
- Она следит за курсором

## Запуск

Из-за ES-модулей страницу нужно открывать через HTTP, не через `file://`:

```bash
cd portfolio
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Как сделать своей

- Тексты секций (обо мне / проекты / навыки / опыт / контакты) — в `index.html`
- Имя, серийный номер, телеметрия — там же
- Цвета и типографика — CSS-переменные в начале `style.css`
- Камера, свет, bloom — константы в `main.js`
