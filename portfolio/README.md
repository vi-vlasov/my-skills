# Portfolio — Detroit: Become Human style

Одностраничное портфолио по мотивам главного меню Detroit: Become Human:
анимированная 3D-девушка, «андроидный» LED-индикатор на виске, тонкое меню
слева, светлая киберпанк-эстетика.

## Технологии рендеринга

- **three.js r170** (лежит локально в `vendor/`, страница полностью автономна)
- Физически корректные материалы (PBR, metalness/roughness)
- HDR image-based lighting (PMREM + RoomEnvironment)
- ACES filmic tone mapping, sRGB output
- Мягкие тени (PCF soft shadow map 2048², shadow radius)
- Постобработка: UnrealBloomPass + OutputPass (EffectComposer)
- Скелетная анимация (AnimationMixer, риггованный glTF-персонаж)
- Cyan rim light + key/fill схема освещения, туман, плавающие частицы
- Параллакс камеры от мыши + «дыхание» камеры
- LED-индикатор — HTML-элемент, привязанный к кости головы через
  проекцию мировых координат в экранные

## Запуск

Из-за ES-модулей страницу нужно открывать через HTTP, не через `file://`:

```bash
cd portfolio
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Как сделать своей

- Тексты секций (обо мне / проекты / навыки / контакты) — в `index.html`
- Имя, серийный номер, статусная строка — там же
- Цвета и типографика — CSS-переменные в начале `style.css`
- Камера, свет, bloom, скорость анимации — константы в начале `main.js`
- Модель можно заменить любым риггованным `.glb` c анимацией
  (положить в `assets/` и поправить путь в `main.js`; масштаб и посадка
  на пол нормализуются автоматически)
